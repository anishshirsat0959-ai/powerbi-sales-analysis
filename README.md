# powerbi-sales-analysis
# Power BI Sales Analysis Project

# Project Overview
This project demonstrates an end-to-end sales data analysis using MySQL and Power BI.

## Data Source
- MySQL database
- Tables: Customers, Products, Orders

## Data Modeling
- Created relationships using customer_id and product_id
- Clean star schema design

## Tools & Technologies
- Power BI
- Power Query
- DAX
- MySQL

## key Analysis
- Total sales by customer
- Sales by product
- Quantity Analysis
- Discount Impact on revenue

## Outcome
Developed an interactive dashboard providing actionable business insights.
